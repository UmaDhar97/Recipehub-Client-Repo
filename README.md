# 🍽️ RecipeHub — Recipe Sharing Platform

A community-driven platform where food enthusiasts can create, share, discover, and manage recipes.

## 🌐 Live Link

https://recipehub-client-repo-hlfbj8ym7-umadhar97s-projects.vercel.app/


## ✨ Features

### User Features
- ✅ Register & Login (Email + Google)
- ✅ Add up to 2 recipes (Free plan)
- ✅ Unlimited recipes (Premium)
- ✅ Like, Favorite & Report recipes
- ✅ Premium membership via Stripe
- ✅ User Dashboard
- ✅ Profile management

### Admin Features
- ✅ Manage Users (Block/Unblock)
- ✅ Manage Recipes (Feature/Delete)
- ✅ Review Reports
- ✅ View Transactions
- ✅ Admin Dashboard with stats

## 🛠️ Tech Stack

### Frontend
- React.js + Vite
- Tailwind CSS
- Framer Motion
- Firebase Authentication
- Stripe Payment
- Axios

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- Stripe API
- ImgBB Image Upload
